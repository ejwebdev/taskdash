# TaskDash

## Overview

A real-time task manager app designed for effortless task creation, seamless editing, and quick deletion.

## Features

-   Responsive design
-   Multi-pages
-   Authentication form
-   Add, Edit, and Delete tasks
-   Drag and Drop tasks

## Technologies Used

-   Tailwind and React
-   Firebase
